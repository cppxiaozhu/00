## 自成一体的软件包

小狼毫输入法是 Windows 下的中州韵前端，这个大家都很熟悉了。在 RIME 的用户群中，98五笔用户是非常特别的，他们这个社群在 RIME 可被形码利用的范围内，做了高度集成的打包。

### 五笔拆分显示

最初较为亮眼的功能点是通过 OpenCC 的注解功能，实现了「拆分显示」：

- ctrl + shfit + H = 控制拆分显隐
- ctrl + shfit + J = 控制注解模式
- z 键 引导「临时拼音」，并提示「上屏历史」
